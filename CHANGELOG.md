# Changelog

本项目遵循语义化版本号。Marketplace 后续版本会在此记录面向用户的变化。

## Unreleased

### Added

- 支持评审 `.c` 文件，并自动配对同名 Markdown 参考讲解。
- 评审按钮显示当前快捷键，并随用户的键位修改自动更新。
- 侧边栏支持查看按文件保存的历史评审列表和完整结果。

### Changed

- 评测历史改为保存在 `.c` 或 `.cpp` 所在目录旁的 `.408judge/` 文件夹，并使用文件名与源码路径哈希区分记录。
- 历史记录入口移至固定底部工具栏，评审内容滚动时始终保持可见，并为后续功能入口预留位置。
- 思考等级选择器移至底部工具栏右侧，在各界面状态下保持固定位置。
- 底部工具栏新增主页和设置入口，可随时返回插件首页或直接打开 408 Judge 设置。
- 右下角选择器支持 Flash、Pro 两个评审模型与三种思考强度的六种组合，并同步保存模型和强度设置。

## 0.11.8 - 2026-07-16

### Added

- 为思考阶段加入分段概括、平滑进入动效和自动跟随滚动。
- 为最终评审结果加入完整内容块揭示动效。
- 为正确结论加入支持“减少动态效果”的庆祝动画。
- 加入独立 Activity Bar 图标和 Marketplace 图标。

### Changed

- 相同思考标题下的概括改为追加，减少标题重复和高频切换。
- 最终结果不再向用户暴露未完成的流式文字。
- 优化侧边栏宽度、长文本换行和结果完成时的布局稳定性。
- 默认 API 请求超时时间由 90 秒延长至 600 秒。

### Fixed

- 修复思考阶段大量重复节点、错误覆盖概括和自动滚动失效。
- 修复侧边栏右侧出现异常空白的问题。
